 📚 Book Store Web Application

  Overview

The **Book Store Web Application** is a full-stack project that allows users to browse, search, and purchase books online. It is built using the **MERN stack (MongoDB, Express, React, Node.js)**.

---

  Features

  User Features

* User registration and login
* Browse all books
* Search books by title or author
* View detailed book information
* Add books to cart
* Place orders

  Admin Features

* Add new books
* Update book details
* Delete books
* Manage orders
* Manage users

---



 1️ Clone the repository

```
git clone https://github.com/your-username/books-store.git
cd books-store
```

---

 2️ Backend Setup

```
cd backend
npm install
npm run dev
```

 Server will run on:

```
http://localhost:5000
```

---

 Frontend Setup

Open a new terminal:

```
cd frontend
npm install
npm start
```

 Frontend will run on:

```
http://localhost:5173/
```
